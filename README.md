# SlimeTinker2

SlimeTinker2 is an unofficial Gugu/Slimefun United based fork of the original SlimeTinker addon.

This build keeps the newer Gugu dependency path, restores the original developer's English wording/lore where possible, preserves DynaTech addon materials, and updates the old InfinityExpansion material hooks to InfinityExpansion2.

## Features kept/restored

- Original SlimeTinker items, recipes, workstations, guide text, and English lore.
- InfinityExpansion2 material support using IE2 item IDs.
- DynaTech addon material support restored.
- SlimefunWarfare, LiteXpansion, TranscEndence, and Networks support kept.
- GitHub Actions build using Java 25 while compiling Java 17-compatible bytecode.

## Requirements

- Paper or compatible fork
- Slimefun / Slimefun United / compatible Gugu Slimefun build
- GuizhanLibPlugin
- InfinityExpansion2 for IE2 materials
- DynaTech if you want DynaTech materials
- Java 17+ bytecode runtime; Java 25 supported

## Notes

Do not install old InfinityExpansion beside InfinityExpansion2. This fork targets InfinityExpansion2.

Original plugin by Sefiraat. This fork is an unofficial compatibility update.
